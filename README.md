# Master Academic Directory Pro

A lightweight, client-side single-file dashboard application designed to log, manage, and calculate academic weighted averages based on standard DepEd multi-component assessment parameters (Written Work, Performance Tasks, and Quarterly Assessments).


##  Key Features
- **Dynamic Hierarchy Sorting:** Automatically bubbles highly active subject channels to the top of the workspace dashboard matrix based on the cumulative data points logged.
- **DepEd Component Weighted Formula Execution:** Seamlessly tracks multi-tier weighted averages, handling precise rounding-up logic natively inside the DOM engine.
- **Multi-Workspace Architecture:** Supports spawning independent, isolated academic directories (e.g., changing semesters or grade tiers) without crossing telemetry parameters.
- **Client-Side Persistence:** Utilizing local browser memory storage (`localStorage`) for zero-latency private caching, equipped with a portable string cloud backup/load system wrapper.

##  Stack Configuration
- **Languages:** HTML5, CSS3 Grid & Variables, Vanilla JavaScript (ES6+ Context)
- **Deployment Platform:** GitHub Pages CDN Pipeline
